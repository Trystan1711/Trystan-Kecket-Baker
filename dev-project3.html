Site Easy2Drive

Développement du site Easy2Drive, une plateforme web permettant de mettre en relation des auto-écoles et des clients. Le front-end a été développé avec le framework Angular, en mettant l’accent sur l’ergonomie, la fluidité de navigation et l’expérience utilisateur. Le projet inclut également la création et l’utilisation d’une API développée en PHP, testée et documentée à l’aide de Postman. Cette API permet la gestion des données (utilisateurs, auto-écoles, offres) et assure la communication entre le front-end et le back-end.
