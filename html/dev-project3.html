Easy2Drive

Développement d'une plateforme web permettant de mettre en relation des auto-écoles et des clients. Le front-end a été développé avec Angular, et le back-end avec une API PHP documentée via Postman.

Catégorie Application web
Stack Angular · PHP · Postman
Type Full Stack

Contexte

L'objectif était de créer une plateforme de mise en relation entre auto-écoles et candidats au permis de conduire. Le projet couvre à la fois le front-end Angular (ergonomie et fluidité) et le back-end PHP (gestion des données, API REST).

Architecture technique

Le front-end Angular gère la navigation entre les vues (liste des auto-écoles, profils, réservation). L'API PHP, testée et documentée avec Postman, assure la communication entre le front et la base de données : gestion des utilisateurs, des auto-écoles et des offres disponibles.
